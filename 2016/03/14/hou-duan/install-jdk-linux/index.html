Linux上安装JDK
Linux Java JDK
后端 2016-03-14 阅读

  1. 上传jdk1.6.0_26.zip

  2. 解压后设置环境变量

    • /etc/rc.local

        export JAVA_HOME=/opt/jdk1.6.0_26
  export JRE_HOME=${JAVA_HOME}/jre
  export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
  export PATH=${JAVA_HOME}/bin:$PATH

    • ~/.bashrc

        export JAVA_HOME=/opt/jdk1.6.0_26
  export JRE_HOME=${JAVA_HOME}/jre
  export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib

  export PATH=${JAVA_HOME}/bin:$PATH
